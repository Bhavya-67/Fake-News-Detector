# Fake-News-Detector

📰 Fake News Detector

A simple and interactive web application that analyzes news text and predicts whether the news is Real or Fake.This project demonstrates how text analysis and machine learning concepts can be used to detect misinformation.

🚀 Live Demo

🔗 Live Application:https://qmwnkmhzjawggkg2cfse9n.streamlit.app/

📌 Project Overview

The Fake News Detector allows users to enter a news headline or article and instantly check whether it is likely Real News or Fake News.

This project is designed as a beginner-friendly AI project and demonstrates how to build and deploy a simple web application using Python.

⚙️ Installation

1️⃣ Clone the repository

git clone https://github.com/yourusername/Fake-News-Detector.git

2️⃣ Navigate to the project folder

cd Fake-News-Detector

3️⃣ Install required libraries

pip install -r requirements.txt

4️⃣ Run the application

streamlit run app.py

▶️ How to Use

Open the application using the live link.

Enter a news headline or article in the text box.

Click Check News.

The system will display whether the news is Real or Fake.
